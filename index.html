使い方ガイド

基本操作

ノード配置 ツールバーのシェイプボタンを選び、キャンバス上でクリック（ドラッグでサイズ指定も可）
テキスト入力 ノードをダブルクリックでテキスト編集。Shift+Enterで改行
テキスト再編集 確定済みのテキスト要素をクリックで選択後、もう一度クリックで再編集
縦書き プロパティパネルの「縦書き」チェックで、ノード・テキスト・レーン名を縦書きに変更
矢印接続 ノードにカーソルを合わせて表示される青い丸●をドラッグし、接続先ノードへドロップ
矢印ツール 矢印ツール選択後、キャンバス上でドラッグするとフリー矢印を描画
テキストツール テキストツール選択後、キャンバスをクリックでテキスト配置
アイコンツール アイコンツール選択後、キャンバスをクリックでアイコン配置

選択・移動

単一選択 要素をクリック
複数選択 Shiftキーを押しながらクリック、またはドラッグで範囲選択
全選択 Ctrl+A
移動 選択した要素をドラッグ。複数選択時は選択範囲内の空白ドラッグでも移動可能
軸固定移動 Shiftキーを押しながらドラッグで水平/垂直に固定

整列・配置

整列ボタン 複数選択時、ツールバーの整列ボタンで左・中央・右・上・中央・下揃えが可能
均等配置 複数選択時、横方向・縦方向の均等配置ボタンで要素間の隙間を均等に分配
ガイド表示 ツールバーのガイドドロップダウンで16:9 / 9:16 / 4:3 / 3:4のアスペクト比ガイドを表示。出力には含まれない

矢印の編集

経路調整 矢印を選択すると表示されるハンドル（白丸）をドラッグで曲げ位置を変更
接続点変更 矢印を選択すると始点・終点にオレンジの四角■が表示。ドラッグで接続位置を変更
ウェイポイント Ctrl+クリックで矢印上に通過点を追加。再度Ctrl+クリックで削除
交差ブリッジ 矢印同士が交差する箇所に半円の跨ぎ（ブリッジ）を自動表示。ツールバーの「ブリッジ」ボタンでON/OFF切替（デフォルトON）

画像

画像配置 画像ファイルをキャンバスにドラッグ&ドロップ
トリミング 画像をクリック → 表示されるメニューから「トリミング」を選択。ハンドルで範囲を調整し、枠外クリックまたはEnterで確定。Escでキャンセル
再トリミング トリミング済みの画像を再度トリミングすると、元画像全体が表示され範囲を広げることも可能（非破壊）
リサイズ 画像選択時の四隅ハンドルをドラッグ。Shiftキーで縦横比を維持
回転 画像選択時の上部緑色ハンドルをドラッグ

プロパティ編集

サイドパネル 要素を選択すると右側パネルに色・線種・透明度などのプロパティが表示される
一括編集 複数選択時はまとめてプロパティ変更が可能
グループ化 複数選択時にポップアップから「グループ化」で要素をまとめられる

レーン管理

レーン追加 左サイドバーの「+ レーン追加」ボタン
レーン名変更 レーン名をダブルクリック。Shift+Enterで改行、Enterで確定
レーン名 縦書き ツールバーの「A横/A縦」ボタンで全レーンの横書き/縦書きを一括切替。個別レーンはプロパティパネルで設定
レーン幅/高さ レーン間の境界をドラッグでリサイズ。縦型モードでは左端レーンの左側ハンドルで左方向にも拡張可能
レーン並替 レーンヘッダーをドラッグ&ドロップで並び替え
レーン方向 ツールバーの「横→縦」ボタンで横型/縦型を切替
ホワイトボード ツールバーの「WB」ボタンでレーンなしのフリーモードに切替

キーボードショートカット

Ctrl+Z 元に戻す
Ctrl+Y やり直し
Ctrl+A 全選択
Ctrl+C コピー
Ctrl+V 貼り付け
Delete / Backspace 選択要素を削除
F4 直前の書式変更を繰り返し（入力中も使用可能）
Shift+Enter テキスト編集中に改行を挿入
Shift+Ctrl+ドラッグ 要素を複製してドラッグ
Shift+ドラッグ 水平/垂直に移動を固定
Ctrl+クリック 矢印・ノードにウェイポイント/編集点を追加・削除
Enter テキスト編集確定 / トリミング確定
Escape トリミングキャンセル / 選択解除

保存・出力

保存 JSONファイルとして保存。読込ボタンまたはドラッグ&ドロップで復元
画像出力 PNG / SVG形式で出力。範囲指定出力も可能
